1. 1번 코드를 수정하여 아래 조건을 만족하는 클라이언트 프로그램을 만드시오.
1.1. 프로그램을 시작하면 10개의 클라이언트 프로세스를 process를 fork한다.
1.2. 각 프로세스는 fork된 순서대로 자신의 ID (1부터 10까지)를 가지도록 한다.
1.3. 각 프로세스는 서버에 접속한다.
1.4. 각 프로세스는 자신의 ID와 100부터 200까지의 정수 중 임의의 수 두 개를 서버에 전송한다.
1.5. 각 프로세스는 서버에 숫자를 보낸 뒤, 서버가 보내온 응답을 받아 자신의 ID와 함 께 출력한다.
1.6. 1.3과 1.4를 10번 반복한 후 서버와의 접속을 끊는다.
1.7. 처음 만든 10개의 프로세스가 모두 종료될 때까지 기다린 후 프로그램을 종료한다.
